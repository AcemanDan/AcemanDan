Self-taught .NET developer building full-stack web applications with ASP.NET Core, Entity Framework Core, and SQL Server. Passionate about clean architecture, practical database design, and shipping real, working software.

## 🚀 Featured Project

### [InvenTrack — Inventory Management System](https://github.com/jamesvarnell/InventoryManagementApp)

> Full-stack inventory management web app built with ASP.NET Core 10 Razor Pages and EF Core 10.

- 📦 Product, category, and supplier CRUD with search, filter, and pagination
- 🔄 Stock transaction logging (Restock / Sale / Adjustment / Return)
- 📊 Chart.js dashboards — inventory value, stock activity, low stock reports
- ⚙️ App-wide settings, admin profile, and database management
- 🗄️ SQL Server LocalDB · Bootstrap 5 · C# 13 · .NET 10
